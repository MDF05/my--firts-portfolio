ABOUT ME

Hai, saya Dava — mahasiswa Teknik Informatika di Universitas Siber Asia (UnSIA) yang suka ngoding sejak lulus SMA. Sebelum kuliah, saya udah duluan terjun ke dunia programming lewat bootcamp Dumbways Indonesia selama hampir setahun, dilanjut program IDCamp × Dicoding fokus di React, dan sekarang lagi ikut program DBS Foundation × Dicoding buat belajar Machine Learning.

Di sisi teknis, saya udah punya pengalaman nyata bikin aplikasi web dengan Next.js, React, Vue.js, dan Laravel. Nggak cuma frontend — saya juga nyentuh backend pakai Node.js, Express, dan Spring Boot, kelola database PostgreSQL, MySQL, dan MongoDB, serta deploy pakai Docker dan Vercel. Sempat juga ngulik mobile development dengan React Native.

Sejak Maret 2025 saya aktif kerja sebagai freelancer — ngerjain proyek web untuk klien sambil terus ningkatin skill. Saya tipe orang yang nggak bisa diam, selalu penasaran sama hal baru, dan senang kalau bisa kolaborasi bareng orang lain. Masih banyak yang mau dipelajari, tapi semangatnya nggak pernah surut. 🚀
